# Language Translation Tool

## Overview
This project is a Language Translation Tool developed using Python. It allows users to translate text through a simple graphical user interface.

## Features
- User-friendly GUI using Tkinter
- English to Tamil translation
- Fast and accurate translation
- Error handling for invalid input

## Technologies Used
- Python
- Tkinter
- Deep Translator

## How to Run
1. Install Python
2. Install required package:
   pip install deep-translator
3. Run:
   python translator.py

   Update README.md
